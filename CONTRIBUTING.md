# Contributing

Thanks for your interest in contributing to OAuth2 Sidecar!

## Ways to Contribute

- Report bugs
- Improve documentation
- Add examples
- Extend provider support

## Development Setup

1. Fork the repository
2. Clone your fork
3. Create a feature branch
4. Make changes with clear commits
5. Open a pull request against `main`

## Guidelines

- Keep changes focused and well-documented
- Add or update examples when changing behavior
- Prefer configuration over code changes where possible
- Run any available tests or validation scripts

## Code of Conduct

Please be respectful and constructive. This project aims to be welcoming and inclusive.
